# Extrator de Dados de PDF para Excel

Este script Python utiliza bibliotecas como `PyMuPDF`, `pytesseract` e `openpyxl` para extrair dados de um arquivo PDF, realizar reconhecimento óptico de caracteres (OCR) em imagens do PDF e salvar os resultados em um arquivo Excel.

## Pré-requisitos

Certifique-se de ter as seguintes bibliotecas instaladas:


pip install python-poppler-qt5
pip install pytesseract
pip install openpyxl
pip install PyMuPDF

Além disso, o Tesseract OCR deve estar instalado no sistema. Você pode baixá-lo em https://github.com/tesseract-ocr/tesseract.

###Observação: 
Este README presume que o script é executado em um ambiente compatível com as bibliotecas e dependências mencionadas.
